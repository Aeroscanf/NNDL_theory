## 第一部分：机器学习与基础模型（重灾区）

### 第二章：机器学习概述
**[本章命题重点]：风险最小化理论的数学等价性、泛化误差分解。**

### Q1: [泛化误差分解] 泛化误差的“偏差-方差”数学分解公式是什么？请解释其物理意义。
**解答**：
期望错误（泛化误差）最终可严谨分解为三部分之和：$R(f) = \text{bias}^2 + \text{variance} + \varepsilon$
1. **偏差平方 (Bias²)**：$E_x[(\mathbb{E}_D[f_D(x)] - f^*(x))^2]$
   - **物理意义**：衡量“所有不同训练集上训练出的平均模型”与“最优真实模型”的平方差。代表学习算法本身的**绝对拟合能力**。
2. **方差 (Variance)**：$E_x[\mathbb{E}_D[(f_D(x) - \mathbb{E}_D[f_D(x)])^2]]$
   - **物理意义**：衡量“特定训练集上的模型”与“平均模型”的离散程度。代表模型对特定训练集波动的**敏感度**（即是否容易发生过拟合）。
3. **噪声 ($\varepsilon$)**：客观存在的不可约减的误差下界。

### Q2: [损失函数] 请默写平方损失、交叉熵损失、Hinge损失的公式，并简述适用场景。
**解答**：
1. **平方损失（Quadratic Loss）**：$L(y, f(x)) = \frac{1}{2}(y - f(x))^2$。**适用场景**：回归任务。由于其梯度随误差减小而减小，利于收敛，但不适用于分类（对异常点过于敏感且非凸）。
2. **交叉熵损失（Cross-Entropy Loss）**：$L(y, f(x)) = - \sum_{c=1}^C y_c \log f_c(x)$。**适用场景**：分类任务（LR, Softmax）。等价于最大似然估计。
3. **Hinge损失（Hinge Loss）**：$L(y, f(x)) = \max(0, 1 - yf(x))$。**适用场景**：SVM二分类。**物理意义**：自带“间隔”属性，分类正确且置信度高时损失严格截断为0，赋予了模型稀疏性（支持向量）。

### Q3: [推导] 为什么说最大似然估计（MLE）等价于经验风险最小化（ERM）？
**解答**：
在线性回归中，假设样本标签 $y$ 服从以模型预测值 $w^T x$ 为均值、$\sigma^2$ 为方差的高斯分布 $\mathcal{N}(w^Tx, \sigma^2)$。
写出似然函数并取负对数（NLL）：
$$NLL = -\log \prod_{n=1}^N \frac{1}{\sqrt{2\pi}\sigma}\exp\left(-\frac{(y^{(n)}-w^Tx^{(n)})^2}{2\sigma^2}\right)$$
化简后：
$$NLL = \frac{1}{2\sigma^2}\sum_{n=1}^N (y^{(n)} - w^T x^{(n)})^2 + C$$
**结论**：最大化似然函数等价于最小化上述 NLL，这在数学结构上**完全等价于采用平方损失函数的经验风险最小化（最小二乘估计 LSE）**。

### Q4: [推导] 请推导最大后验估计（MAP）是如何等价于带有L2正则化的结构风险最小化（SRM）的？
**解答**：
在MLE基础上，贝叶斯学派假设参数 $w$ 本身是一个随机向量，并服从均值为 0 的各向同性高斯先验分布 $p(w) = \mathcal{N}(0, \nu^2 I)$。
根据贝叶斯公式最大化后验概率：$p(w|X, y) \propto p(y|X, w) p(w)$
两边取自然对数：
$$\log p(w|X, y) \propto \log p(y|X, w) + \log p(w)$$
代入高斯先验的对数：$\log p(w) = -\frac{1}{2\nu^2} \|w\|_2^2 + C$
目标函数变为最大化：$-\frac{1}{2\sigma^2} \|y - X^Tw\|_2^2 - \frac{1}{2\nu^2} \|w\|_2^2$
**结论**：等价于最小化平方损失的经验风险的同时，加上了一个 $\lambda\|w\|_2^2$ 的 **$L_2$ 正则化项**（其中正则化系数 $\lambda = \sigma^2/\nu^2$）。这就是岭回归的本源。

### Q5: [PAC理论] 默写PAC学习理论的核心公式，并解释“可能”与“近似正确”的统计学定义。
**解答**：
核心公式：**$P\left( R(f) - R_{emp}^D(f) \le \epsilon \right) \ge 1 - \delta$**
1. **近似正确**：泛化错误与经验错误的差值小于一个微小界限 $\epsilon$（只要在这个界限内，我们就认为学到的假设逼近了真实情况）。
2. **可能**：由于训练集是随机采样的，永远无法100%保证上述界限成立。因此引入概率阈值 $1 - \delta$，即有 $1-\delta$ 的概率学习到这个近似正确的假设。

### Q6: [评价指标] 默写Precision, Recall, F1公式。并说明微平均和宏平均的区别。
**解答**：
对于类别 $c$：
- **Precision（查准率）**：$P_c = \frac{TP_c}{TP_c + FP_c}$ （预测为 $c$ 中，正确预测的比例）
- **Recall（查全率）**：$R_c = \frac{TP_c}{TP_c + FN_c}$ （真实为 $c$ 中，正确预测的比例）
- **F1值**：$F1 = \frac{2PR}{P+R}$
**区别**：
- 宏平均（Macro Average）：先算各类的 P/R，再求算术平均。**平等对待每个类别**。
- 微平均（Micro Average）：先将所有类的 TP/FP/FN 加和，再算总体 P/R。**平等对待每个样本**。

### Q7: [模型评估] 极度不平衡样本下，为何 Accuracy 会失效，而 AUC 依然可靠？
**解答**：
- **Accuracy 失效**：在 1:99 的极度失衡中，分类器只要盲猜全为负类，就能获得 99% 的准确率。Accuracy 被庞大的负类“绑架”，失去对少数类召回能力的评估意义。
- **AUC 的可靠性本质**：ROC 曲线的横坐标 FPR（$\frac{FP}{FP+TN}$）和纵坐标 TPR（$\frac{TP}{TP+FN}$）是**在正类群体内部和负类群体内部独立计算的**。因此，负类样本规模的剧烈扩张只会影响 FPR 内部的基数，不会跨界干扰 TPR，ROC 曲线不会发生扭曲。

### Q8: [无免费午餐定理] 简述没有免费午餐定理（NFL）的核心内涵。
**解答**：
NFL 定理证明：对于基于迭代的最优化算法，如果不对数据分布做任何假设，所有算法在全部可能问题上的平均性能是完全相同的（甚至和纯随机搜索一样差）。**物理意义**：不存在脱离具体任务的“万能模型”，强调了“具体问题必须具体分析”（Inductive Bias 归纳偏置的重要性）。

### 第三章：线性模型
**[本章命题重点]：分类器损失函数的等价推导、支持向量机KKT条件、LR与SVM的深层对比。**

### Q1: [LR的线性证明] 逻辑回归引入了非线性的Sigmoid函数，为什么仍被称为线性模型？（用公式证明）
**解答**：
1. **对数几率视角**：后验概率 $p(y=1|x) = \sigma(w^T x)$。正反例概率比值称为几率，取对数：
   **$\log \frac{p(y=1|x)}{p(y=0|x)} = \log(\exp(w^T x)) = w^T x$**
   结论：LR是在用特征的线性组合去精确拟合类别标签的“对数几率”。
2. **决策边界视角**：令 $p(y=1|x) = 0.5$，即 $\frac{1}{1+\exp(-w^Tx)} = 0.5 \implies \exp(-w^Tx) = 1 \implies w^T x = 0$。
   结论：其决策边界 $w^Tx=0$ 在高维空间中是一个标准的**线性超平面**。

### Q2: [LR损失推导] LR交叉熵损失函数的统计学推导（基于伯努利分布的MLE）。
**解答**：
假设真实标签 $y \in \{0, 1\}$，模型预测概率为 $\hat{y} = p(y=1|x) = \sigma(w^Tx)$。
巧妙利用伯努利分布，将条件概率统合成一个公式：
$$p(y|x) = \hat{y}^y (1 - \hat{y})^{(1-y)}$$
根据极大似然估计，对所有样本的联合概率取负对数（Negative Log-Likelihood, NLL）：
$$NLL = -\log \prod p(y^{(n)}|x^{(n)}) = -\sum_{n=1}^N \log \left( \hat{y}^{(n) y^{(n)}} (1 - \hat{y}^{(n)})^{(1-y^{(n)})} \right)$$
利用对数性质展开：
**$$NLL = - \sum_{n=1}^N \left[ y^{(n)} \log \hat{y}^{(n)} + (1-y^{(n)}) \log(1-\hat{y}^{(n)}) \right]$$**
此公式在数学形式上与信息论中的交叉熵公式完全一致。

### Q3: [Softmax推导] Softmax回归的参数冗余性数学证明及解决方法。
**解答**：
- **数学证明**：Softmax预测概率为 $p(y=c|x) = \frac{\exp(w_c^T x)}{\sum_i \exp(w_i^T x)}$。
  若所有类别权重向量同时减去一个常数向量 $v$，新的权重为 $w_i' = w_i - v$：
  **$p'(y=c|x) = \frac{\exp((w_c - v)^T x)}{\sum_i \exp((w_i - v)^T x)} = \frac{\exp(-v^T x) \exp(w_c^T x)}{\exp(-v^T x) \sum_i \exp(w_i^T x)} = p(y=c|x)$**
  证明了目标函数存在无数个最优解。
- **解决方法**：在实际训练中，往往必须加入**正则化惩罚项（如权重衰减 Weight Decay）**，打破参数空间的平坦“山谷”，强迫模型收敛到范数最小的唯一最优解。

### Q4: [感知机] 写出感知机的损失函数形式及权重更新公式；感知机为何无法解决XOR问题？
**解答**：
- **损失函数**：$L(w) = \max(0, -y w^T x)$ （$y \in \{+1, -1\}$）
- **权重更新**：随机梯度下降，当分类错误（$y w^T x < 0$）时，更新 **$w \leftarrow w + yx$**。
- **无法解决XOR问题的几何证明**：感知机的决策边界是一条严格的直线（或超平面）。在二维空间中，异或（XOR）的四个点呈现对角线交叉分布（正例在(0,1)和(1,0)，反例在(0,0)和(1,1)）。数学上绝对不可能画出一条单一的直线将对角线两侧的正反例完美分割。

### Q5: [SVM推导] 默写SVM的最大间隔优化目标公式演进。
**解答**：
1. 样本到超平面的几何间隔为 $\gamma^{(n)} = \frac{y^{(n)}(w^Tx^{(n)}+b)}{\|w\|}$。目标是最大化全局最小间隔：$\max_{w,b} \min_n \frac{y^{(n)}(w^Tx^{(n)}+b)}{\|w\|}$。
2. 令最小的函数间隔 $y^{(n)}(w^Tx^{(n)}+b) = 1$（这构成了支持向量），优化目标等价变形为：
   **$\min_{w,b} \frac{1}{2} \|w\|^2$**
   **$\text{s.t. } y^{(n)}(w^T x^{(n)} + b) \ge 1, \quad \forall n$**
3. 引入松弛变量 $\xi$（软间隔）：
   $\min_{w,b} \frac{1}{2} \|w\|^2 + C \sum_{n=1}^N \xi_n$
   $\text{s.t. } y^{(n)}(w^T x^{(n)} + b) \ge 1 - \xi_n, \quad \xi_n \ge 0$

### Q6: [SVM KKT] 默写拉格朗日乘数法及KKT条件，并解释互补松弛条件对支持向量的物理意义。
**解答**：
构造拉格朗日函数：$L(w,b,\lambda) = \frac{1}{2}\|w\|^2 + \sum_{n=1}^N \lambda_n(1 - y^{(n)}(w^Tx^{(n)}+b))$
**核心KKT条件**：
1. 梯度为0：$\nabla_w L = 0, \nabla_b L = 0 \implies w = \sum \lambda_n y^{(n)} x^{(n)}, \sum \lambda_n y^{(n)} = 0$
2. 主问题约束：$1 - y^{(n)}(w^Tx^{(n)}+b) \le 0$
3. 对偶变量非负：$\lambda_n \ge 0$
4. **互补松弛条件：$\lambda_n (1 - y^{(n)}(w^Tx^{(n)}+b)) = 0$**
**物理意义**：如果样本不在间隔边界上（即严格大于1），则必有 $\lambda_n = 0$，对 $w$ 无贡献；只有当样本恰好位于间隔边界上（括号内为0）时，$\lambda_n > 0$。这些使得 $\lambda_n > 0$ 的样本就是**支持向量**，它说明SVM的决策边界**仅由极少数支持向量绝对主导**。

### Q7: [深度对比] LR与SVM在损失函数和决策边界主导样本上的深度对比。
**解答**：
1. **从损失函数角度**：
   - **SVM 使用 Hinge Loss**：自带“Margin”属性。只要样本被正确分类且置信度够高（函数间隔 $\ge 1$），损失就**严格截断为0**，不再产生梯度。
   - **LR 使用 Cross-Entropy Loss**：一条平滑的渐近线。即使样本分类正确，损失也**永远大于0**。LR会“贪得无厌”地继续压榨正确样本，试图把概率推向1。
2. **从对决策边界的影响角度**：
   - **SVM**：由**少数支持向量绝对主导**。在远离边界处增删大量正确样本，超平面纹丝不动。
   - **LR**：由**全局样本共同拉扯决定**。每一个样本都会通过梯度 $\sum x^{(n)}(y^{(n)} - \hat{y}^{(n)})$ 贡献残差，大量多数类样本会导致边界偏移。

### Q8: [核技巧] 为什么SVM广泛使用核函数（Kernel Trick），而LR较少使用？
**解答**：
SVM的最终决策函数 $f(x) = \text{sgn}(\sum_{n=1}^N \lambda_n y^{(n)} x^{(n)T} x + b)$ 完全依赖于样本之间的**内积**运算。通过将内积 $\langle \phi(x), \phi(z) \rangle$ 替换为核函数 $K(x, z)$，SVM可以隐式地在高维甚至无限维空间计算，且极其高效（因为大部分 $\lambda_n=0$，只需算支持向量）。
而LR的决策依赖于全局所有样本，不存在这种稀疏性。如果对LR强行使用核技巧，计算所有样本间核矩阵的复杂度是 $O(N^3)$，在海量数据下算力无法承受。

## 第二部分：基础模型（反向传播与CNN/RNN）

### 第四章：前馈神经网络
**[本章命题重点]：激活函数求导、BP矩阵链式求导、梯度消失机理。**

### Q1: [激活函数] 常见激活函数的公式及导数公式默写（Sigmoid, Tanh, ReLU）。
**解答**：
1. **Sigmoid**: $f(x) = \frac{1}{1+e^{-x}}$ | 导数：**$f'(x) = f(x)(1 - f(x))$**
2. **Tanh**: $f(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$ | 导数：**$f'(x) = 1 - f(x)^2$**
3. **ReLU**: $f(x) = \max(0, x)$ | 导数：**$f'(x) = 1$ (当 $x>0$)，$0$ (当 $x<0$)**

### Q2: [死亡ReLU] 什么是死亡ReLU问题？如何解决？
**解答**：
- **定义**：在训练时，如果参数发生不恰当的更新，导致某个 ReLU 神经元对于所有训练数据输入都小于0，那么它的输出恒为0，导数也恒为0。在以后的BP过程中，**梯度无法回传，该神经元的参数永远无法更新**，相当于在网络中“死亡”。
- **解决方法**：使用 **Leaky ReLU** ($f(x) = \max(\gamma x, x)$，$\gamma$ 是微小常数) 或 **PReLU** (参数化的 $\gamma$)，使得在 $x<0$ 时保留微小的非零梯度。

### Q3: [新型激活函数] Swish和GELU的作用及机制。
**解答**：
- **Swish**: $f(x) = x \cdot \sigma(\beta x)$。它是一种“自门控”激活函数，在逼近ReLU的同时，在原点附近保持了平滑和微小的非单调性，利于深层网络优化。
- **GELU (Gaussian Error Linear Unit)**: $f(x) = x \cdot \Phi(x)$，其中 $\Phi(x)$ 是标准正态分布的累积概率。它结合了Dropout的随机正则化思想和ReLU的门控机制，目前是Transformer/BERT的标配。

### Q4: [通用近似定理] 简述通用近似定理的内容与意义。
**解答**：
- **内容**：对于任何一个定义在实数空间有界闭集上的连续函数，存在一个具有线性输出层和至少一个隐藏层（使用“挤压”性质或ReLU等激活函数）的**前馈神经网络，只要其隐藏神经元的数量足够多，就可以以任意的精度去近似这个函数**。
- **意义**：给出了神经网络作为“万能函数逼近器”的理论保证，证明了通过增加神经元，网络理论上具备学习任何复杂映射的能力。

### Q5: [硬核推导] 误差反向传播算法（BP）中，隐藏层误差项 $\delta^{(l)}$ 的递推矩阵公式。
**解答**：
定义第 $l$ 层净输入为 $z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}$，误差项 $\delta^{(l)} = \frac{\partial L}{\partial z^{(l)}}$。
根据链式法则：$\frac{\partial L}{\partial z^{(l)}} = \frac{\partial a^{(l)}}{\partial z^{(l)}} \cdot \frac{\partial z^{(l+1)}}{\partial a^{(l)}} \cdot \frac{\partial L}{\partial z^{(l+1)}}$
- $\frac{\partial z^{(l+1)}}{\partial a^{(l)}} = (W^{(l+1)})^T$
- $\frac{\partial a^{(l)}}{\partial z^{(l)}} = \text{diag}(f'_l(z^{(l)}))$
得出核心递推公式：
**$$\delta^{(l)} = f'_l(z^{(l)}) \odot ((W^{(l+1)})^T \delta^{(l+1)})$$**

### Q6: [硬核推导] 默写BP中权重 $W$ 和偏置 $b$ 的梯度公式。
**解答**：
利用上一题算出的本层误差项 $\delta^{(l)}$：
- **权重梯度**：**$\frac{\partial L}{\partial W^{(l)}} = \delta^{(l)} (a^{(l-1)})^T$**  *(维度: $m^{(l)} \times m^{(l-1)}$)*
- **偏置梯度**：**$\frac{\partial L}{\partial b^{(l)}} = \delta^{(l)}$**

### Q7: [梯度消失] 为什么深层网络容易出现梯度消失？请从数学公式严谨说明。
**解答**：
根据BP算法，误差项的反向传播展开：
$\delta^{(l)} = \prod_{i=l}^{L-1} \left( \text{diag}(f'(z^{(i)})) (W^{(i+1)})^T \right) \delta^{(L)}$
如果网络使用 Sigmoid 作为激活函数，其导数 $f'(z) = \sigma(z)(1-\sigma(z))$ 的**最大值仅为 0.25**。
在链式法则的连乘中，每一层都要乘上一个小于 0.25 的数。当网络层数 $L-l$ 很深时，多项连乘会导致 $\delta^{(l)}$ 以指数级速度迅速衰减至0。此时靠近输入层的参数无法获得梯度更新，这就是梯度弥散（消失）的数学本质。

### Q8: [网络优化] 在高维非凸优化中，比局部最优点更难逃离的是什么？
**解答**：
**鞍点（Saddle Point）**。
在高维空间中，一个点在每一维上同时是最低点（局部最优）的概率是极小的。大部分梯度为 0 的点都是鞍点（在某些维度是最高点，在某些维度是最低点）。梯度下降法在鞍点附近会接近停滞，难以逃离。

### 第五章：卷积神经网络
**[本章命题重点]：维度与参数量计算、各种卷积核的物理意义。**

### Q1: [特性本质] 简述CNN中局部连接和权重共享特性，及其对参数量的影响公式。
**解答**：
- **局部连接**：图像空间具有局部性，卷积层每个神经元只与上一层的一个局部窗口（感受野）相连，无需全连接。
- **权重共享**：自然图像具有平移不变性，对于同一种特征提取器（卷积核），在输入图像的所有位置滑动时，共享同一组参数。
- **参数量影响**：假设输入 $D$ 通道，输出 $P$ 通道，卷积核大小 $K \times K$。全连接参数量与特征图大小有关；而CNN参数量骤降为：**$(K \times K \times D + 1) \times P$**（$1$ 为偏置），与输入图像大小完全无关！

### Q2: [公式默写] 写出卷积层输出特征图维度的精确计算公式。
**解答**：
给定输入图像一维的长度 $W$（或高度 $H$），卷积核大小 $K$，滑动步长 $S$ (Stride)，两端零填充数量 $P$ (Padding)。
输出特征图的长度 $W_{out}$ 为：
**$$W_{out} = \lfloor \frac{W - K + 2P}{S} \rfloor + 1$$**

### Q3: [反向传播] 汇聚层（Pooling）的反向传播梯度是如何计算的？
**解答**：
汇聚层没有可学习参数，但需将误差项 $\delta^{(l+1)}$ 传递给上一层 $\delta^{(l)}$。
1. **Max Pooling**：误差只回传给上一层感受野中**值最大**的神经元，其余神经元梯度分配为 0。（类似路由路由选择）。
2. **Mean Pooling**：误差**平均分配**给上一层感受野中的所有神经元，即把 $\delta^{(l+1)}$ 乘以 $1/(K \times K)$，散布回上一层。

### Q4: [1x1卷积] 1x1卷积核的核心作用是什么？（极其重要）
**解答**：
1x1 卷积（Network in Network）不会改变特征图的宽高，其核心作用有三：
1. **跨通道信息交互**：在深度（通道）维度上进行线性组合，实现多通道特征的融合。
2. **降维/升维（通道缩放）**：通过控制1x1卷积核的数量（输出通道数 $P$），可以灵活地压缩或增加通道维度，极大降低后续大卷积核（如 3x3）的计算量（FLOPs）。
3. **增加非线性**：1x1 卷积后通常紧跟 ReLU，在不增加空间感受野的前提下，增加了网络的非线性表达能力。

### Q5: [ResNet原理] 为什么ResNet（残差网络）能解决深度网络退化问题？写出其恒等映射公式。
**解答**：
- **恒等映射公式**：**$H(x) = f(x; \theta) + x$** （$f(x)$ 为残差函数，$x$ 为直连跨层边 Shortcut）。
- **解决退化问题机理**：深度网络退化不是由于过拟合，而是由于深层非线性网络极难拟合“恒等映射 $H(x)=x$”。ResNet 将目标转换为拟合残差 $f(x) = H(x) - x$。当网络层已饱和时，优化器只需将 $f(x)$ 的权重压迫为0，即可轻松实现恒等映射。此外，直连边 $x$ 在反向传播时提供了无损的梯度高速公路（导数为1），彻底缓解了梯度消失。

### Q6: [网络结构] 简述Inception模块的结构特点。
**解答**：
Inception（GoogLeNet核心）的特点是**“拓宽网络宽度”和“多尺度特征提取”**。
它在同一层并联了多个不同大小的卷积核（1x1, 3x3, 5x5）和最大汇聚层，提取多尺度的空间特征，最后在通道维度上拼接（Concatenate）。同时在 3x3 和 5x5 卷积前，巧妙使用了 1x1 卷积进行降维，控制了计算复杂度。

### Q7: [转置卷积] 转置卷积（反卷积）与微步卷积的数学本质是什么？
**解答**：
- **数学本质**：在全连接网络中前向是乘 $W$，反向是乘 $W^T$。卷积操作可等价表示为极度稀疏的仿射矩阵 $C \cdot x$。转置卷积并非卷积的严格逆运算，而是**在数学形式上乘以转置矩阵 $C^T$ 的一种映射**，用于将低维特征图映射回高维特征图（如图像生成、语义分割）。
- **微步卷积**：当转置卷积的步长 $S > 1$ 时（相当于正向卷积的 $S < 1$），在实现上是通过在输入特征图的相邻元素之间**插入 $S-1$ 个零**，再进行标准卷积，从而实现上采样扩维。

### Q8: [空洞卷积] 空洞卷积（Dilated Convolution）的作用及感受野计算。
**解答**：
- **作用**：在不增加模型参数数量和计算量的前提下，通过在卷积核元素间插入“空洞”，**指数级扩大感受野**。同时避免了传统 Pooling 层下采样带来的空间分辨率丢失。
- **计算**：假设原卷积核大小 $m$，膨胀率（Dilation Rate）为 $d$，则等效的实际卷积核大小变为：
  **$m' = m + (m-1) \times (d-1)$**。然后将 $m'$ 代入标准输出公式计算。

### 第六章：循环神经网络
**[本章命题重点]：BPTT展开、LSTM公式默写与门控意义。**

### Q1: [BPTT] BPTT（随时间反向传播）与一般BP的区别是什么？为什么会梯度爆炸/消失？
**解答**：
- **区别**：BPTT 将 RNN 按照时间步 $T$ 展开，视为一个极深的共享参数的前馈网络。在计算某时刻梯度的时，必须累加过去所有时刻对该参数的偏导数。
- **数学原因**：误差项反向传播时存在连乘积 $\delta_{t,k} = \prod_{i=k}^{t-1} \left( \text{diag}(f'(z_i)) U^T \right) \delta_{t,t}$。
  其中 $U$ 是隐状态循环权重。如果 $\| \text{diag}(f')U^T \| > 1$，长时间间隔连乘会导致**梯度爆炸**；如果 $<1$，多步连乘会导致**梯度消失**（即当前状态很难对距离久远的历史状态参数进行有效梯度回传，产生长期依赖问题）。

### Q2: [优化技巧] 梯度爆炸的解决方法（公式）。
**解答**：
使用**梯度截断（Gradient Clipping）**。当梯度的 $\ell_2$ 范数超过阈值 $b$ 时，强制将其按比例缩放：
如果 $\|g\|_2 > b$，则 **$g \leftarrow \frac{b}{\|g\|_2} g$**。以此防止参数更新过大摧毁模型。

### Q3: [核心默写] 默写LSTM的核心结构与五大数学公式（极高频）。
**解答**：
输入 $x_t$，上一隐藏状态 $h_{t-1}$，上一细胞状态 $c_{t-1}$：
1. **遗忘门（Forget Gate）**：决定丢弃多少旧记忆。
   **$f_t = \sigma(W_f x_t + U_f h_{t-1} + b_f)$**
2. **输入门（Input Gate）**：决定吸纳多少新记忆。
   **$i_t = \sigma(W_i x_t + U_i h_{t-1} + b_i)$**
3. **输出门（Output Gate）**：决定输出多少到当前隐状态。
   **$o_t = \sigma(W_o x_t + U_o h_{t-1} + b_o)$**
4. **细胞状态更新（Cell State）**：线性状态转移，候选状态 $\tilde{c}_t = \tanh(W_c x_t + U_c h_{t-1} + b_c)$。
   **$c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t$**
5. **隐状态更新（Hidden State）**：
   **$h_t = o_t \odot \tanh(c_t)$**

### Q4: [LSTM机理] LSTM是如何通过线性自环（细胞状态 $c_t$）缓解梯度消失的？
**解答**：
在传统RNN中，$h_t$ 到 $h_{t-1}$ 的梯度包含了 $U$ 矩阵的连乘。
而在 LSTM 中，记忆的核心通道是**内部状态 $c_t$**。其更新公式 $c_t = f_t \odot c_{t-1} + \dots$ 是一种**加法更新机制**（线性关系）。在反向传播求导时，$\frac{\partial c_t}{\partial c_{t-1}} \approx f_t$。如果遗忘门 $f_t \approx 1$（保持打开），梯度就能沿着这条“线性高速公路”几乎无损地传导回任意长的历史时刻，完美绕过了矩阵 $U$ 连乘导致的梯度消失！

### Q5: [GRU] 简述GRU的结构特点及与LSTM的对比。
**解答**：
- **结构特点**：GRU 将输入门和遗忘门合并为单一的**更新门 $z_t$**，并去掉了独立的细胞状态 $c_t$，引入**重置门 $r_t$** 控制对历史状态的利用。
  公式核心：$h_t = z_t \odot h_{t-1} + (1 - z_t) \odot \tilde{h}_t$
- **对比**：GRU 相比 LSTM 少了一个门，参数量减少了约 25%，计算效率更高，在多数任务上性能与 LSTM 相当。

### Q6: [Seq2Seq] Seq2Seq模型在NLP中的三种模式是什么？
**解答**：
1. **序列到类别（N to 1）**：例如文本分类。序列输入，取最后一个状态 $h_T$ 或平均状态池化后接入分类器。
2. **同步序列到序列（N to N）**：例如词性标注（POS）。每个时间步同步有一个输入和一个输出，严格等长。
3. **异步序列到序列（Encoder-Decoder）**：例如机器翻译。输入长度 $S$ 与输出长度 $T$ 不同。先将源序列编码为语义向量 $c$，再由 Decoder 自回归解码。

### Q7: [RNN vs CNN] 为什么在处理变长序列时RNN天然优于标准前馈网络？
**解答**：
前馈网络要求输入维度严格固定，且难以跨时间步共享特征提取能力。
RNN通过**状态的自反馈循环**，在时间维度上做到了**权重共享**。无论输入序列多长，RNN只需一套参数 $(U, W, b)$ 就能递归处理，且隐状态 $h_t$ 能累积前面所有的变长历史信息。

### Q8: [双向RNN] 为什么需要双向循环神经网络 (Bi-RNN)？
**解答**：
单向 RNN 的隐状态 $h_t$ 仅能捕获时刻 $t$ 及其**之前的历史上下文**。但在诸如命名实体识别、词性标注等任务中，当前词的语义很大程度上取决于它**未来的下文**。Bi-RNN 分别使用正向和逆向的两个独立 RNN 处理序列，最终的隐状态为 $h_t = [\overrightarrow{h_t} \oplus \overleftarrow{h_t}]$，完美捕获了全局的上下文信息。

### 第七章：网络优化与正则化
**[本章命题重点]：优化器公式辨析、BN与Dropout核心机理。**

### Q1: [优化器] SGD、Momentum、Nesterov加速梯度的公式对比及物理意义。
**解答**：
1. **标准 SGD**：$\theta_t = \theta_{t-1} - \alpha g_t$。只看当前梯度，易在“峡谷”地带震荡。
2. **动量法 (Momentum)**：引入物理学惯性，$\Delta \theta_t = \rho \Delta \theta_{t-1} - \alpha g_t$。积累历史梯度移动平均。**物理意义**：在方向一致的维度加速，在方向交替改变（震荡）的维度减速抵消，平滑更新轨迹。
3. **Nesterov (NAG)**：$\Delta \theta_t = \rho \Delta \theta_{t-1} - \alpha g_t(\theta_{t-1} + \rho \Delta \theta_{t-1})$。先按动量“超前”探一步，在**超前点**计算梯度。**物理意义**：增加了一层“预见性”，在快要冲过极小值时能提前反向拉回，稳定性更好。

### Q2: [自适应学习率] AdaGrad、RMSprop、Adam 更新公式区别与优缺点。
**解答**：
1. **AdaGrad**：累积所有历史梯度平方 $G_t = \sum_{\tau} g_{\tau} \odot g_{\tau}$，步长变为 $\frac{\alpha}{\sqrt{G_t+\epsilon}}$。**缺点**：后期 $G_t$ 爆炸导致学习率极速衰减至0，过早停止。
2. **RMSprop**：改用指数衰减移动平均 $G_t = \beta G_{t-1} + (1-\beta)g_t \odot g_t$。**优点**：解决了AdaGrad学习率单调趋近0的问题，适合处理非平稳目标。
3. **Adam**：融合了 Momentum (一阶矩 $M_t$) 和 RMSprop (二阶矩 $G_t$)。
   公式：$\theta_t = \theta_{t-1} - \frac{\alpha}{\sqrt{\hat{G}_t} + \epsilon} \hat{M}_t$ （带偏差修正）。**优点**：鲁棒性极高，通常是NLP/CV首选的自适应优化器。

### Q3: [参数初始化] Xavier初始化推导的假设前提和方差公式推导过程。
**解答**：
- **假设前提**：激活函数在其零点附近近似线性（如 Tanh, Sigmoid）；输入特征与权重相互独立且均值为0。
- **推导目标**：为了使得前向传播中方差不被放大或缩小（$var(z) = var(x)$），以及反向传播时梯度方差稳定，需满足：
  正向要求：$n_{in} var(w) = 1$；反向要求：$n_{out} var(w) = 1$。
- **折中结果**：令 $var(w) = \frac{2}{n_{in} + n_{out}}$。如果采用均匀分布 $w \sim U[-r, r]$，其方差为 $r^2/12$。解得初始化的截断边界为：**$r = \sqrt{\frac{6}{n_{in} + n_{out}}}$**。

### Q4: [BN推导] 默写Batch Normalization (BN) 的完整公式（包含缩放和平移）；训练与测试区别？
**解答**：
设当前小批量中某一神经元净输入集为 $B = \{z^{(1)}, \dots, z^{(K)}\}$。
1. 计算均值与方差：$\mu_B = \frac{1}{K}\sum z^{(k)}$, $\sigma_B^2 = \frac{1}{K}\sum (z^{(k)} - \mu_B)^2$
2. 零均值单位方差归一化：$\hat{z}^{(k)} = \frac{z^{(k)} - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}$
3. **缩放和平移变换**（核心恢复表达能力）：**$y^{(k)} = \gamma \hat{z}^{(k)} + \beta$**
**测试阶段区别**：测试时由于往往每次只输入单一样本，无法计算批量均值方差。须使用训练时利用**指数移动平均 (EMA)** 记录的全局均值 $\mu$ 和全局方差 $\sigma^2$ 代替当前的 $\mu_B, \sigma_B^2$。

### Q5: [BN机理] 为什么BN能加速网络训练？什么是内部协变量偏移（Internal Covariate Shift）？
**解答**：
- **内部协变量偏移 (ICS)**：在深层网络训练中，底层权重的微小改变会经过层层放大，导致高层神经元输入的分布剧烈改变。高层不得不重新适应这种分布变化，导致学习困难和收敛极慢。
- **BN的加速机理**：BN 强行将每个隐藏层神经元的净输入分布“拉回”到均值为 $\beta$、方差为 $\gamma^2$ 的稳定状态。这彻底解耦了层与层之间的参数相互牵制，缓解了ICS。同时，它使得数值落在激活函数（如Sigmoid）的不饱和（线性）区，有效避免了梯度消失。

### Q6: [Dropout] Dropout的公式及训练与测试阶段的缩放操作。
**解答**：
- **公式**：以概率 $p$ 将神经元置 0（被丢弃），以概率 $1-p$ 保持原值。$y = f(W d(x) + b)$，其中掩码 $m \sim \text{Bernoulli}(1-p)$。
- **阶段差异缩放**：在训练阶段，神经元有 $p$ 的概率停止工作，使得激活的期望按比例缩小了。为了保证在**测试阶段**（全开状态）时，下一层接收到的激活期望值与训练时一致，测试时必须将所有神经元的输出**乘以 $(1-p)$**；（或者在现代框架的实现中，即在训练阶段直接将保留的值除以 $(1-p)$，测试时不作处理，称为 Inverted Dropout）。

### Q7: [Dropout解释] 从集成学习和贝叶斯角度解释Dropout防过拟合的机制。
**解答**：
- **集成学习角度**：对于包含 $N$ 个神经元的网络，Dropout 相当于在其 $2^N$ 种子网络中进行采样。每次 Mini-batch 都在训练一个不同的子网络。最终预测等价于对指数级数量的、参数共享的模型做平均（Ensemble），从而显著降低了模型方差。
- **贝叶斯解释**：Dropout 在数学上被证明近似等价于对网络参数的高斯先验进行贝叶斯变分推断（Variational Inference）。相当于对参数做蒙特卡罗采样，赋予了模型一定的不确定性约束。

### Q8: [Label Smoothing] 标签平滑的公式及其防止过拟合的机制。
**解答**：
- **公式**：在原本绝对的 one-hot 标签（正确类为1，错误类为0）中混入微小噪声 $\epsilon$。
  平滑后正确类的目标概率：$1 - \epsilon + \frac{\epsilon}{K}$；错误类的目标概率：$\frac{\epsilon}{K}$ （$K$ 为类别总数）。
- **防过拟合机制**：在标准的 Cross-Entropy 中，为了让正确类概率无限逼近 1，模型会拼命压榨错误类，使权重的绝对值越来越大（极易过拟合错误标注或噪点）。Label Smoothing 相当于**提供了一种软目标**，阻止模型做出“过于自信”的绝对预测，强迫类间特征留有缓冲余地。

### 第八章：注意力机制与外部记忆
**[本章命题重点]：Attention核心打分函数、Self-Attention核心推导。**

### Q1: [Attention类型] 软注意力（Soft Attention）和硬注意力（Hard Attention）的区别。
**解答**：
- **软注意力**：计算所有输入信息的概率分布 $\alpha_i$，最后对所有输入进行**加权平均**：$att = \sum \alpha_i x_i$。它是全程平滑、可微的，可以直接通过标准反向传播（BP）算法进行端到端训练。
- **硬注意力**：基于概率分布 $\alpha_i$，只**最高概率采样出单一的一个输入信息**。由于采样操作产生了离散的阶跃，不可微，无法用标准BP计算梯度，必须借助**强化学习（REINFORCE，策略梯度）**来进行训练。

### Q2: [打分函数] 默写三种常见的注意力打分函数（加性、点积、缩放点积）公式。
**解答**：
给定查询向量 $q$ 和输入向量 $x_i$：
1. **加性模型（Additive）**：$s(x_i, q) = v^T \tanh(W x_i + U q)$ （参数为 $v, W, U$）
2. **点积模型（Dot-Product）**：$s(x_i, q) = x_i^T q$ （要求两者维度相同）
3. **缩放点积模型（Scaled Dot-Product）**：$s(x_i, q) = \frac{x_i^T q}{\sqrt{d}}$ （$d$ 为向量维度）

### Q3: [缩放点积] 为什么点积注意力在维度较高时需要除以 $\sqrt{d_k}$ 进行缩放？
**解答**：
若 $q$ 和 $k$ 分别服从均值为 0、方差为 1 的相互独立的标准高斯分布。其点积 $q \cdot k = \sum_{i=1}^d q_i k_i$ 的结果服从均值为 0，**方差为 $d$** 的分布。
当维度 $d_k$ 很大时，点积结果的方差会极大。这些数量级庞大的数值喂入 Softmax 层后，会导致概率分布极度“尖锐化”（即某个值趋近于1，其余趋近于0），从而**落入 Softmax 的饱和区，导致梯度极度消失**。除以 $\sqrt{d_k}$ 强行把方差拉回 1，使 Softmax 梯度稳定。

### Q4: [Self-Attention] 默写自注意力机制的 $Q, K, V$ 计算公式及其物理意义。
**解答**：
对于输入序列矩阵 $X$：
- $Q = W_Q X$（查询向量 Queries，代表寻找目标时的主动方）
- $K = W_K X$（键向量 Keys，代表被查询信息本身的属性标识）
- $V = W_V X$（值向量 Values，代表被查询信息蕴含的真实内容）
**统一计算公式**：
**$$\text{Self-Att}(Q,K,V) = \text{softmax}\left(\frac{K^T Q}{\sqrt{d_k}}\right) V$$**
**物理意义**：打破了序列的空间距离限制，序列中每一个元素都动态地与序列中的所有其它元素计算相关性（通过 $Q \cdot K$），进而融合全序列所有的 $V$ 形成全局表示。

### Q5: [复杂度对比] 相比RNN和CNN，Self-Attention处理长序列的复杂度与并行性优势？
**解答**：
- **并行性**：RNN 是严苛的 O(N) 时序串行计算，无法并行；CNN 和 Self-Attention 的特征提取在整个序列维度上高度解耦，能利用 GPU 实现 **O(1) 的完美并行计算**。
- **复杂度与长程依赖**：
  - RNN 捕捉跨度为 $N$ 的长程依赖需要经过 $N$ 次传递，极易发生信息丢失（梯度消失）。
  - CNN 受限于感受野大小，需要极深的网络或大空洞率才能覆盖长距离。
  - **Self-Attention** 任意两个词直接做点积，信息传递的**最长路径永远是 O(1)**，完美解决了极其长程的依赖问题。

### Q6: [Multi-Head] 多头注意力（Multi-Head Attention）的计算逻辑是什么？
**解答**：
利用多个不同的投影矩阵组 $(W_Q^m, W_K^m, W_V^m)$ 平行地对输入进行多次不同的子空间映射，产生 $M$ 个互相独立的“头”（Head）。
$$\text{head}_m = \text{self-attention}(Q_m, K_m, V_m)$$
最后将所有 Head 提取到的表示在通道维度上**拼接（Concatenation）**，并经过一个线性变换矩阵 $W_O$ 投影为最终输出：
$$\text{MultiHead} = W_O [\text{head}_1 \oplus \text{head}_2 \dots \oplus \text{head}_M]$$
**目的**：使得模型能够在不同表征子空间中捕捉更加多元化的特征（如同多通道CNN提取各种纹理）。

### Q7: [Pointer Network] 指针网络是如何利用注意力机制的？
**解答**：
在传统的Seq2Seq任务中，输出是一个固定词表内的词。但在求解诸如TSP问题、排序问题时，输出序列的元素必须是**输入序列中元素的位置索引**。
指针网络修改了Attention的用途：它**抛弃了 Attention 中的最后一步加权平均（即不计算 $\sum \alpha_i v_i$）**。它直接截取 Softmax 归一化后生成的注意力分布 $\alpha$，将其作为一种“软性指针”，用来**指示当前解码步应输出原序列中哪个位置的下标**。

### Q8: [Attention本质] Attention机制是否可以看作多层感知机（MLP）的等价物？
**解答**：
Attention 本质上可以视为一种**具备动态生成权重（Dynamic Weights）功能的全连接层**。
在标准全连接网络（MLP）中，权重矩阵是静态的，不仅无法处理变长输入，而且每次输入不同的句子用的也是同一套僵化的权重；自注意力模型中，权重 $\alpha_{ij}$ 是根据输入特征 $q_i, k_j$ 的实时相似度**动态计算**出来的，这极大增强了模型的泛化与序列适应能力。

## 第九章 & 第十章：无监督学习与模型独立学习
**[本章命题重点]：PCA推导、Bagging与Boosting辨析、领域适应、MAML机制。**

### Q1: [PCA推导] 简述PCA最大投影方差推导过程（拉格朗日乘子法求特征值分解）。
**解答**：
1. 目标：将数据 $X$ 投影到方向 $w$ 上，使得投影后的方差最大化。设样本协方差矩阵为 $S = \frac{1}{N} X X^T$。投影后的方差为 **$w^T S w$**。
2. 约束：为了防止方差无穷大，约束投影向量的模为 1，即 **$w^T w = 1$**。
3. 构造拉格朗日函数：$L(w, \lambda) = w^T S w + \lambda(1 - w^T w)$。
4. 对 $w$ 求偏导并令为 0：$S w - \lambda w = 0 \implies \mathbf{S w = \lambda w}$。
**结论**：这在数学上完全等价于求解协方差矩阵 $S$ 的**特征值分解方程**。欲使方差 $\lambda$ 最大，对应的最优投影方向 $w$ 即为协方差矩阵 $S$ 中**最大特征值对应的特征向量**。

### Q2: [自编码器] 自编码器（AE）和降噪自编码器（DAE）的目标函数。
**解答**：
- **AE (Auto-Encoder)**：模型分为编码器 $z=f(x)$ 与解码器 $x'=g(z)$。其目标是极小化输入与输出之间的**重构误差**，目标函数：**$L = \sum_n \| x^{(n)} - g(f(x^{(n)})) \|^2$**。
- **DAE (Denoising Autoencoder)**：为了增强鲁棒性，人为给输入引入噪声（如 Dropout 置 0 或加高斯噪声）得到被破坏的 $\tilde{x}$。网络不仅要从 $\tilde{x}$ 编码出 $z$，还必须强迫解码器复原出**未被破坏的**纯净 $x$。目标函数：**$L = \sum_n \| x^{(n)} - g(f(\tilde{x}^{(n)})) \|^2$**。

### Q3: [集成学习] 集成学习中Bagging与Boosting的区别（从偏差和方差的视角）。
**解答**：
- **Bagging（如随机森林）**：通过自助采样（Bootstrap）生成平行且独立的弱训练集。由于基模型互相独立平行投票，因此其主要作用是**极大地降低模型方差（Variance）**，防止过拟合，故基模型常选用容易过拟合的高方差模型（如未剪枝深层决策树）。
- **Boosting（如AdaBoost, GBDT）**：后一个模型串行地试图纠正前一个模型犯下的错误。因此其主要作用是**极大地降低模型偏差（Bias）**，增强整体拟合能力，故基模型常选用拟合能力弱的浅层模型（如浅层决策树/树桩）。

### Q4: [AdaBoost] 简述AdaBoost的样本权重更新逻辑与分类器权重计算。
**解答**：
1. **分类器权重**：若当前弱分类器 $f_m$ 的加权错误率为 $\epsilon_m$，则其在最终投票中的绝对话语权为 $\alpha_m = \frac{1}{2} \log \frac{1-\epsilon_m}{\epsilon_m}$。错误率越小，权重越大。
2. **样本权重更新**：新一轮样本权重按照 $w_i^{(m+1)} \propto w_i^{(m)} \exp(-\alpha_m y_i f_m(x_i))$ 更新。如果样本被分错（$y_i f_m(x_i) < 0$），指数项 $>1$，其**权重会被大幅放大**；迫使下一个弱分类器对这些“难啃的骨头”投入更高的注意力。

### Q5: [迁移学习] 归纳迁移学习与转导迁移学习（领域适应）的区别。
**解答**：
- **归纳迁移（Inductive Transfer）**：源领域与目标领域往往**输入空间相同，但任务不同**（$P_s(y|x) \neq P_t(y|x)$）。典型做法：源域有海量有标数据（如ImageNet），预训练后在目标域精调（Fine-tuning）。
- **转导迁移（Transductive Transfer）**：又叫领域适应（Domain Adaptation）。源域与目标领域**任务完全相同，但输入数据分布不同**（比如真实照片域 vs 手绘动漫域，$P_s(x) \neq P_t(x)$）。典型做法：在训练阶段直接对齐源域与目标域特征空间的边缘分布。

### Q6: [领域适应对抗] 基于对抗的领域适应（Domain Adaptation）原理是什么？
**解答**：
为了解决由于协变量偏移（源域与目标域边缘分布不一致）导致性能骤降，构建一个**领域判别器（Domain Discriminator）**。
- 领域判别器：拼命判断当前提取的特征是属于源域还是目标域。
- 特征提取器（生成器）：拼命“欺骗”领域判别器，使自己提取的特征**模糊掉两个领域的差异**（即提取出 Domain-Invariant 表示）。当判别器无法区分时，在这个公共特征空间上训练的分类器就能在测试集（目标域）无损应用。

### Q7: [元学习] 元学习（Meta-Learning）特别是 MAML 的核心思想是什么？
**解答**：
- **核心思想**：“Learn to Learn” 学习如何学习。旨在利用已有的大量历史任务去总结出一套通用的学习规律，用于快速攻克缺乏数据的新任务。
- **MAML (Model-Agnostic Meta-Learning)**：不特指某种网络架构。它的目标是寻找一组**极佳的全局网络初始化参数 $\theta$**。保证对于任何从未见过的新任务，这组 $\theta$ 只要经过一到两次最简单的梯度下降精调，就能立即收敛到新任务的最优解。它是针对“参数敏感度”的一种二阶梯度优化。

### Q8: [终生学习] 什么是终生学习中的灾难性遗忘（Catastrophic Forgetting）？弹性权重巩固（EWC）是如何解决的？
**解答**：
- **灾难性遗忘**：神经网络在学习新任务 B 的知识时，梯度更新会极其粗暴地冲刷掉（覆盖）模型曾经为历史任务 A 刻苦学习到的参数，导致网络把以前的技能彻底忘光。
- **EWC 解决思路**：计算出一个 Fisher 信息矩阵，用来刻画每个参数对老任务 A 的“重要程度”。在学习新任务 B 时加入一个二次惩罚项：**如果某个参数对老任务很重要，EWC 就像弹簧一样死死拉住它，不允许大范围更新；而对老任务无关紧要的参数，则允许大范围探索去迎合新任务**。

## 第三部分：进阶模型 (选背与拓宽)

### 第11章：概率图模型
### Q1: [PGM概念] 简述有向图模型与无向图模型的条件独立性差异。
**解答**：
- **有向图（贝叶斯网络）**：用有向非循环图描述。边代表**直接因果关系**。其核心局部马尔可夫性为：给定父节点，每个变量独立于它的非后代节点。
- **无向图（马尔可夫随机场）**：边代表**相互对称的依赖关系**。其核心局部马尔可夫性为：给定邻居节点，一个变量独立于图中所有其它变量。如果两个无向节点不直接相连，且它们中间的所有通路都被已知节点阻断，则必然条件独立。

### Q2: [无向图分解] 默写最大团与能量函数（玻尔兹曼分布）的公式。
**解答**：
在无向图中，联合概率分布无法按链式法则直接分解，只能通过最大团（Clique，内部所有点全连通的子图集）的势能函数连乘。根据 Hammersley-Clifford 定理：
$$P(x) = \frac{1}{Z} \prod_{c \in C} \exp(-E_c(x_c))$$
其中 $C$ 是最大团集合，$E_c(x_c)$ 为能量函数，$Z = \sum_x \prod \exp(-E_c(x_c))$ 是保证概率总和为 1 的归一化配分函数。这种形式又称为玻尔兹曼分布。

### Q3: [推断] 隐马尔可夫模型（HMM）属于哪类图模型？
**解答**：
HMM属于**包含隐变量的动态有向概率图模型**。其隐藏状态序列构成一阶马尔可夫链，观测序列由对应的隐藏状态决定（发射概率）。

### 第12章 & 第13章：深度信念与生成模型
### Q4: [RBM] 简述受限玻尔兹曼机（RBM）的结构及对比散度（CD）算法思想。
**解答**：
- **结构**：RBM 是一种二分无向图模型。分可见层 $v$ 和隐藏层 $h$。核心限制是：**同层内节点完全没有连接**，仅存在异层节点间的全连接。这种结构让隐变量在给定可见层时相互独立。
- **对比散度（CD-k）**：传统 Gibbs 采样需要无穷多步才能达到热平衡算配分函数梯度，代价过高。CD 算法用真实样本初始化可见层，仅在 $v \to h \to v'$ 之间进行 $k$ 步来回采样（通常 $k=1$ 即可），用重构出的 $v'$ 近似替代热平衡状态。

### Q5: [VAE] 变分自编码器（VAE）的证据下界（ELBO）推导与重参数化技巧。
**解答**：
- **ELBO**：由于边际似然 $\log p(x)$ 包含积分无法求解，VAE引入变分分布 $q(z|x)$ 近似隐变量后验。
  $\log p(x) = \text{ELBO} + D_{KL}(q(z|x) || p(z|x))$。因为KL散度非负，故最大化似然转化为最大化下界：
  **$\text{ELBO} = E_{z \sim q}[\log p(x|z)] - D_{KL}(q(z|x) || p(z))$**
- **重参数化 (Reparameterization)**：由于隐变量 $z$ 是从高斯分布 $N(\mu, \sigma^2)$ 中采样的，不可导。将采样剥离为噪声 $\epsilon \sim N(0,I)$，则 **$z = \mu + \sigma \odot \epsilon$**。这使得 $\mu$ 和 $\sigma$ 的梯度通道彻底贯通，能够用标准 BP 端到端训练。

### Q6: [GAN] GAN的极小极大博弈目标函数及模式坍塌（Mode Collapse）的KL散度解释。
**解答**：
- **Minimax公式**：
  $\min_G \max_D E_{x \sim p_{data}}[\log D(x)] + E_{z \sim p(z)}[\log(1 - D(G(z)))]$
- **Mode Collapse解释**：当判别器达到最优时，GAN等价于最小化真实分布 $P_r$ 和生成分布 $P_g$ 之间的JS散度。从变种目标函数推导，它被严重等效为**逆向KL散度 $D_{KL}(P_g || P_r)$**。
  逆向 KL 散度特点：当 $P_r$ 为空但 $P_g$ 不为空（生成了假废品）时惩罚无穷大；当 $P_r$ 有真品但 $P_g$ 为空（模型没生成）时惩罚极小。因此，生成器会极度保守，**只敢在最稳妥的一个真实模式峰值点附近无限重复造假**，导致生成样本毫无多样性（模式坍塌）。

### 第14章 & 第15章：深度强化学习与序列生成
### Q7: [MDP与强化学习] 状态值函数 $V(s)$ 与状态-动作值函数 $Q(s,a)$ 的贝尔曼方程默写。
**解答**：
- **$V(s)$ (状态值)**：站在当前状态，期望未来能获得多少总折扣奖励。
  **$V^\pi(s) = E_{a \sim \pi, s' \sim p}[r(s,a,s') + \gamma V^\pi(s')]$**
- **$Q(s,a)$ (Q函数)**：站在当前状态且**已经决定强制做一个特定动作 $a$** 后，期望未来获得的总奖励。
  **$Q^\pi(s,a) = E_{s' \sim p}[r(s,a,s') + \gamma E_{a' \sim \pi}[Q^\pi(s',a')]]$**

### Q8: [DQN] DQN中解决非平稳目标和数据相关性的两大核心技术（经验回放与目标网络冻结）。
**解答**：
1. **经验回放（Experience Replay）**：将智能体游走的每步 $(s,a,r,s')$ 塞进内存池。训练时随机打乱抽取 Batch。**打破了马尔可夫时序轨迹的强烈数据自相关性**。
2. **目标网络冻结（Target Network）**：计算 TD Target 损失时，作为“预测标杆”的 Q 网络参数 $\hat{\theta}$ 临时冻结，隔一段时间才把当前在线更新的 $Q$ 权重 $\theta$ 拷贝过去。**避免了“狗咬自己尾巴”导致的标杆乱晃震荡问题**。

### Q9: [Policy Gradient] 策略梯度定理的核心公式及REINFORCE中的表现形式。
**解答**：
策略梯度不借助 Q 值遍历，直接在策略 $\pi_\theta(a|s)$ 参数空间通过梯度上升。
核心推导将求导放进期望内部，借助 $\nabla \log P = \frac{\nabla P}{P}$ 技巧：
**$\nabla_\theta J(\theta) = E_{\tau}\left[ \left(\sum_{t} \nabla_\theta \log \pi_\theta(a_t|s_t)\right) G(\tau) \right]$**
**物理意义**：直接增加高回报 $G(\tau)$ 轨迹所对应动作序列的采样对数概率。

### Q10: [序列生成] 简述束搜索（Beam Search）和曝光偏差（Exposure Bias）。
**解答**：
- **束搜索（Beam Search）**：为了防止每步取绝对最大概率导致的贪婪次优解，每步解码时**始终保留并扩展前 $K$ 个**联合概率累积最大的候选序列（$K$为Beam Size）。
- **曝光偏差与计划采样**：训练RNN自回归生成时，前一步的输入总是“强制喂入真实的金标（Teacher Forcing）”；测试时，前一步的输入却是模型“自己生成的（可能有错的）结果”。这导致分布偏差和错误雪崩。
  **计划采样（Scheduled Sampling）**：在训练中逐步引入自己生成的错词作为下一步输入，锻炼模型自身的纠错适应力。

### Q1: [最大间隔目标] 请推导并默写 SVM 从“几何间隔”到“最大间隔优化目标”的数学演进公式，并说明什么是支持向量？
**解答**：
1. **几何间隔定义**：数据集中每个样本 $x^{(n)}$ 到分割超平面 $w^Tx + b = 0$ 的有向距离（几何间隔）为：
   **$$\gamma^{(n)} = \frac{y^{(n)}(w^Tx^{(n)}+b)}{\|w\|}$$**
2. **全局最小间隔**：定义整个数据集中所有样本到超平面的最短距离为间隔 $\gamma = \min_n \gamma^{(n)}$。支持向量机的目标就是寻找一个超平面使得 $\gamma$ 最大：
   $$\max_{w,b} \gamma \quad \text{s.t.} \quad \frac{y^{(n)}(w^Tx^{(n)}+b)}{\|w\|} \ge \gamma, \forall n$$
3. **等价化简（核心考点）**：由于同时缩放 $w$ 和 $b$ 不改变超平面的位置，我们可以令 $\|w\| \cdot \gamma = 1$。此时最大化 $\frac{1}{\|w\|}$ 等价于最小化 $\frac{1}{2}\|w\|^2$，优化目标演进为标准的凸优化问题：
   **$$\min_{w,b} \frac{1}{2} \|w\|^2$$**
   **$$\text{s.t. } y^{(n)}(w^T x^{(n)} + b) \ge 1, \quad \forall n$$**
4. **物理意义**：数据集中所有满足 $y^{(n)}(w^Tx^{(n)}+b) = 1$ 的样本点，就是**支持向量（Support Vector）**。

### Q2: [参数学习与对偶问题] 默写 SVM 的拉格朗日函数，并写出目标函数关于 $w$ 和 $b$ 求导后的结果表达式。
**解答**：
为了求解上述带约束的凸优化问题，引入拉格朗日乘数 $\lambda_n \ge 0$。
1. **构造拉格朗日函数**：
   **$$\Lambda(w, b, \lambda) = \frac{1}{2}\|w\|^2 + \sum_{n=1}^N \lambda_n \left( 1 - y^{(n)}(w^Tx^{(n)}+b) \right)$$**
2. **偏导数求解**：分别对 $w$ 和 $b$ 求偏导并令其等于 0，得到极值点条件：
   * 关于 $w$ 的导数：**$$w = \sum_{n=1}^N \lambda_n y^{(n)}x^{(n)}$$**
   * 关于 $b$ 的导数：**$$0 = \sum_{n=1}^N \lambda_n y^{(n)}$$**
3. **推论**：将 $w$ 的表达式代回原函数，就可以得到拉格朗日对偶函数 $\Gamma(\lambda)$，从而将主问题转化为通过最大化对偶函数来求解 $\lambda^*$ 的问题。

### Q3: [决策函数与核技巧] 写出求解后 SVM 的最终决策函数。为什么 SVM 可以极其方便地引入核函数（Kernel Function）？
**解答**：
1. **最终决策函数**：在求出最优的 $\lambda^*$ 和 $b^*$ 后，将其与 $w = \sum_{n=1}^N \lambda_n^* y^{(n)}x^{(n)}$ 结合，决策函数变为：
   **$$f(x) = \text{sgn}(w^{*T}x + b^*) = \text{sgn} \left( \sum_{n=1}^N \lambda_n^* y^{(n)}(x^{(n)})^Tx + b^* \right)$$**
2. **物理意义与核技巧的引入**：
   从上述公式可以看出，SVM 的最终决策函数**只依赖于新样本 $x$ 与训练样本 $x^{(n)}$ 之间的内积操作 $(x^{(n)})^Tx$**。
   因此，我们可以隐式地将样本从原始特征空间映射到更高维的空间，直接用核函数 $K(x^{(n)}, x) = \phi(x^{(n)})^T\phi(x)$ 替换内积。这使得 SVM 能高效解决非线性可分问题，而无需显式计算极高维度的 $\phi(x)$。

### Q4: [软间隔与损失函数] 为了处理线性不可分数据，SVM 如何引入松弛变量？请证明“软间隔SVM”在数学上完全等价于“Hinge Loss + L2正则化”。
**解答**：
1. **引入松弛变量（Soft Margin）**：为了容忍部分不满足约束的样本，引入松弛变量 $\xi_n \ge 0$，优化问题变为：
   **$$\min_{w,b} \frac{1}{2}\|w\|^2 + C\sum_{n=1}^N \xi_n$$**
   **$$\text{s.t. } 1 - y^{(n)}(w^Tx^{(n)}+b) - \xi_n \le 0, \quad \xi_n \ge 0, \quad \forall n$$**
   其中 $C>0$ 是用来控制间隔和松弛变量惩罚平衡的超参数。
2. **等价 Hinge Loss 推导**：
   上面的约束条件可以改写为 $\xi_n \ge 1 - y^{(n)}(w^Tx^{(n)}+b)$，同时又要求 $\xi_n \ge 0$。因此在求最小化时，$\xi_n$ 的最优取值就是两者中的最大值，即：
   $\xi_n = \max(0, 1 - y^{(n)}(w^Tx^{(n)}+b))$。
   将此式代回目标函数，原带约束的优化问题完美等价于无约束的经验风险最小化形式：
   **$$\min_{w,b} \sum_{n=1}^N \max \left( 0, 1 - y^{(n)}(w^Tx^{(n)}+b) \right) + \frac{1}{C}\frac{1}{2}\|w\|^2$$**
3. **物理意义**：公式的前半部分正是大名鼎鼎的 **Hinge 损失函数**，后半部分的 $\frac{1}{C}\frac{1}{2}\|w\|^2$ 则是标准的 **$L_2$ 正则化项**。
