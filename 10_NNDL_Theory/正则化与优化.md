## 1. 网络正则化 (Regularization)

### 1.1 丢弃法 (Dropout)
**核心思想**：在训练过程中，对每一层神经元以概率 $p$ 随机“丢弃”（置零），相当于训练了无数个共享参数的子网络（集成学习）。

*   **训练阶段 (Training)**：
    *   **操作**：随机生成掩码 $\mathbf{m} \in \{0, 1\}^d$（服从伯努利分布），将输入 $\mathbf{x}$ 与掩码按元素相乘。
    *   **公式**：$y = f(\mathbf{W}(\mathbf{x} \odot \mathbf{m}) + \mathbf{b})$
*   **测试阶段 (Testing)**：
    *   **操作**：保留所有神经元（不丢弃）。
    *   **权重缩放**：为了保证输出期望与训练时一致，需要将输出（或权重）乘以概率 $p$（或者在训练时除以 $p$ 进行**反向 Dropout**）。

> [!example] Python 伪代码 (标准 Dropout)
> ```python
> class Dropout:
>     def __init__(self, p=0.5):
>         self.p = p
>
>     def forward(self, x, mode='train'):
>         if mode == 'train':
>             # 1. 生成掩码 (1代表保留，0代表丢弃)
>             self.mask = np.random.binomial(1, self.p, size=x.shape)
>             # 2. 训练时：随机失活
>             return x * self.mask
>         elif mode == 'test':
>             # 3. 测试时：全保留 + 权重缩放 (维持期望一致)
>             return x * self.p
> ```

### 1.2 批量归一化 (Batch Normalization, BN)
**解决问题**：**内部协变量偏移 (Internal Covariate Shift)**。
即深层网络参数更新导致每一层的输入分布不断变化，导致网络难以训练（需降低学习率、对初始化敏感）。

*   **核心机制**：
    1.  **标准化**：将当前 Batch 的数据归一化为均值 0、方差 1 的分布。
        $$ \hat{z} = \frac{z - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}} $$
    2.  **缩放与平移**：引入可学习参数 $\gamma, \beta$ 恢复网络的表达能力（如果需要的话，网络可以自己学回原始分布）。
        $$ z' = \gamma \hat{z} + \beta $$
*   **训练 vs 测试**：
    *   **训练**：使用当前 **Batch** 的均值和方差。
    *   **测试**：使用**全局**移动平均的均值和方差（估算整个数据集的统计量）。

## 2. 优化算法 (Optimization Algorithms)

### 2.1 优化难点
*   **鞍点 (Saddle Point)**：在高维非凸优化中，梯度为 0 的点大多是鞍点（一维上升、一维下降），而非局部极小值。这是优化的主要障碍。
*   **梯度消失/爆炸**：深层网络中梯度连乘导致。

### 2.2 算法演进谱系

| 算法           | 核心机制        | 特点与考点                                                                                            |
| :----------- | :---------- | :----------------------------------------------------------------------------------------------- |
| **SGD**      | 随机梯度下降      | 随机采样一个样本计算梯度。收敛慢，易震荡。                                                                            |
| **Momentum** | **动量** (惯性) | 引入历史梯度的加权平均。**抑制震荡，加速收敛**（冲过鞍点）。<br>公式：$\Delta \theta_t = \rho \Delta \theta_{t-1} - \alpha g_t$ |
| **Nesterov** | **预判**      | "先走一步，再看梯度"。在动量预判的位置计算梯度，校正更准确。                                                                  |
| **AdaGrad**  | **自适应学习率**  | 累积历史梯度平方和作为分母。适合稀疏数据。**缺点：学习率衰减过快，易提前停止**。                                                       |
| **RMSprop**  | **移动平均**    | 改进 AdaGrad。将累计平方和改为**指数加权移动平均**。解决了学习率过早消失的问题。                                                   |
| **AdaDelta** | 无需学习率       | 不依赖全局学习率，利用参数更新差值的移动平均来确定步长。                                                                     |
| **Adam**     | **集大成者**    | **Momentum (一阶矩) + RMSprop (二阶矩)**。<br>同时兼顾方向的平滑和步长的自适应。                                         |

> [!tip] 考点总结：Adam 的组成
> Adam 算法可以看作是 **Momentum** 和 **RMSprop** 的结合：
> 1.  利用 **Momentum**（一阶矩估计）平滑梯度方向。
> 2.  利用 **RMSprop**（二阶矩估计）自适应调整每个参数的学习率。
> 3.  增加了**偏差修正**（Bias Correction），解决初始化为 0 导致的初期偏差。

### 2.3 梯度截断 (Gradient Clipping)
*   **目的**：解决**梯度爆炸**问题（常见于 RNN）。
*   **方法**：当梯度的模 $||\mathbf{g}|| > \text{threshold}$ 时，将梯度按比例缩小，使其模等于阈值。
